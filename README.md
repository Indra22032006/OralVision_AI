# Oral Cancer Detection using MobileNetV2

This project uses a MobileNetV2-based deep learning model to classify oral images for potential signs of oral cancer.

## Model
- Transfer Learning (MobileNetV2)
- TensorFlow / Keras

## Dataset
- ~1200 images

## How to Run
Open the notebook in Google Colab and run all cells.

## Disclaimer
This is for educational purposes only and not for medical diagnosis.
